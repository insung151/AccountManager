공용계좌 SMS  자동 전송 어플리케이션
====================================
은행사에서 sms로 보내주는 계좌정보를 미리 설정한 다른사람들에게 자동으로 전달한다.
또 특정 기간을 선택해 카카오톡으로 공유할 수도 있다.(카카오톡은 자동 전송 API를 제공하지 않아 카카오톡을 자동으로 전달하는것은 불가능하다)
## 개발환경

## 구현 기능
1. 은행사별 SMS 자동 전달 기능
2. 특정 기간 사용내역 카카오톡 전송 기능
